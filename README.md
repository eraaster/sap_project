# 2025 숭실대학교 Co-op SAP Track Project

## 📌 프로젝트 개요
본 프로젝트는 SAP ABAP을 활용하여 구매 프로세스 전반을 구현한 실습 프로젝트이다.  
구매처 생성(BP)부터 구매오더(PO), 입고(GR), 송장(IV) 처리까지의 흐름을 하나의 시스템으로 구성하였다.

## 🎯 개발 목적
- SAP 구매 프로세스(BP → PO → GR → IV)에 대한 이해
- ABAP 기반 데이터 처리 및 ALV 화면 구현
- Custom Table 설계 및 트랜잭션 흐름 구현 경험

## 🛠 개발 환경
- SAP NetWeaver AS ABAP
- ABAP (Report / ALV Grid)
- Custom Table (Z* 테이블)
